== Description ==

**Product Brands For Woocommerce** Provide to Custom Brand Taxonomy in Woocommerce to setup in your Store. Product Brand is taxonomy of woocommerce to make seperate brand into it.
it make Brand title, description and brand image.

It will show in Single product page in woocommerce after add to cart button.

> **Features**
>
> * Easy Setup
> * Enable/disable plugin option
> * Woocommerce Custom Taxonomy Brands
> * as well brand slider show in woocommerce all page
> * Enable/disable autoplay option
> * Autoplay speed customize option
> * Number Of Item in slider desktop,tablet and mobile options
> * Autoplay stop on focus speed Enable/Disable option
> * Enable/disable show brand title in slider option
> * Enable/disable Show Brand Item In Single Page option
> * Enable/disable Show Brand Item In Shop Page option
> * Customize Single Product Page Brand Position option
> * Add to cart text and background color option
> * Show Slider or Loop in Brand for shortcode option
> * Shop and single page brands view options
> * Brands image size option
> * Brands title text customize option



> **Shortcodes**
>
> * 1) get all brand then shorcode use [product-brands]
> * 2) brand wise product get then shorcode use [product-brands brands="your term slug"] ex.[product-brands brands="tesing-1"]
> * 3) how many number of post then add post per page in shorcode like ex.[product-brands brands="tesing-1" post_per_page=4]
